# Algebra helper

Помогает решать следующие задачи:
1. Сравнения степени n по модулю:
    ```
    f(x) = ax^n + bx^(n-1) + ... + c = 0 (mod m)
    ```
2. Сравнения первой степени с неизвестной:
    ```
    ax = b (mod m)
    ```
3. Системы сравнений первой степени:
    ```
    { ax = b (mod m1)
    { cx = d (mod m2)
    { ...
    ```
